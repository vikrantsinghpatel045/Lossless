How to add a Lossless Track

01

Open the Portal

Head to the Contribute Portal and log in with your GitHub account. The portal handles forking, branching, and Pull Requests for you automatically.

02

Upload Your Audio

Upload a new audio file (.flac, max 99MB). We only support FLAC format for lossless quality.

03

Add Metadata

Provide the Song Title and Artist Name so the track can be indexed properly in the library.

04

Listen in App

Once merged, you can listen to the lossless audio tracks directly from the web or app.
